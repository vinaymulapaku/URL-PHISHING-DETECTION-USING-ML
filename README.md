# URL-PHISHING-DETECTION-USING-ML

## URL Phishing Detection

PhishGuard is a lightweight machine-learning powered tool that helps you detect suspicious or malicious URLs. Whether you're a casual user, developer, or cybersecurity enthusiast, PhishGuard assists in identifying harmful links and keeping you safe online.

---

## ✨ Features

* 🔍 **URL Safety Checker** – Instantly analyze any URL for phishing patterns.
* 🧠 **Smart ML Predictions** – Uses trained ML models such as Random Forest, Logistic Regression, and XGBoost.
* ⚡ **Fast & Lightweight** – Minimal latency and quick results.
* 📊 **Feature-Based Analysis** – Lexical, statistical, and structural URL features.
* 📱 **API-Friendly** – Easy integration with apps and extensions.

## 🚀 Tech Stack

* **Python:** 3.x
* **Libraries:** Pandas, NumPy, Scikit-Learn, XGBoost, Joblib
